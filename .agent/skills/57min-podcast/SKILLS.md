- name: 57min-podcast
  description: |
    고정된 팟캐스트/토크쇼 진행 순서를 엄격히 따르며 음악·스팟광고 시간을 고려한 대본을 생성합니다.
    실제 글자 수를 카운트한 뒤 목표 시간(예: 57분)에 부족하면 자동으로 자연스러운 토크 문단(대화 턴)을 추가하여 맞춤.
    한국어 자연스러운 말하기 속도와 호흡을 반영한 현실적 파라미터 적용.
    
    **고정 진행 순서 (총 목표 시간에 맞춰 자동 조정)**:
    1. 오프닝 음악 + 간단 소개 (총 30초 FIX)
    2. 오늘의 이야기 주제 소개 + 초대 손님/출연자 소개 + 세부 주제 예고
    3. 스팟광고 30초 (FIX)  ← 1회
    4. 소주제 1 토크
    5. 음악 1 (시간 확정)
    6. 소주제 2 토크
    7. 음악 2 (시간 확정) + 스팟광고 30초 (FIX)  ← 2회
    8. 소주제 3 토크
    9. 음악 3 (시간 확정) + 스팟광고 30초 (FIX)  ← 3회
    10. 소주제 4 토크 + 크로징 멘트 + 스팟광고 30초 (FIX)  ← 4회
    11. 엔딩 음악 4 (시간 확정: 전체 목표 시간에 맞춰 페이드아웃 처리)
    
    **추가 워크플로우 (Google Antigravity 연계 추천)**:
    - 대본 완성 후 Antigravity IDE로 이동 가능
    - Antigravity 에이전트에게 "이 텍스트를 새 Google Docs 문서로 생성하고 제목을 '[프로그램명] - [에피소드 제목] 57분 원고'로 설정해줘" 프롬프트 입력
    - 에이전트가 Google Drive에 새 Docs 생성 → 텍스트 붙여넣기 → 링크 반환 (브라우저 자동화 또는 MCP Toolbox 활용)
    - 수동 복붙 대안: 최종 script를 복사 → Google Docs에 직접 붙여넣기
  inputs:
    - total_time: 전체 방송 목표 시간 (예: "57:00" 또는 3420초)  # 문자열 또는 초 단위 숫자 지원
    - opening_intro_time: 오프닝 음악 + 소개 총 시간 (default: 30초 FIX)
    - spot_ad_time: 각 스팟광고 시간 (default: 30초 FIX, 총 4회 삽입 → 120초)
    - music_list: 삽입할 음악 목록 (최소 4곡 추천, 순서대로 배치)
      type: array of objects
      example:
        - title: "오프닝 테마 음악"
          duration: "0:30"
        - title: "아이유 - 나의 옛날 이야기"
          duration: "3:30"
        - title: "혁오 - 톤즈 오브 라이트"
          duration: "3:30"
        - title: "잔나비 - 꿈과 책과 힘과 벽"
          duration: "3:30"
        - title: "엔딩 테마 음악"
          duration: "variable"  # 마지막은 페이드아웃으로 자동 조정
    - participants: 출연자 역할 목록
      type: array of strings
      example: ["주영철 교수(MC)", "박정근 교수(게스트)", "김사무관(게스트)"]
      default: ["주영철 교수(MC)", "박정근 교수(게스트)"] # 평소 구성. 특별한 날 김사무관(게스트, 이름 수정 예정) 추가 가능
    - style: 진행 스타일
      enum: ["대화체/수다형", "정보전달형", "혼합형", "인터뷰형"]
      default: "대화체/수다형"
    - speaking_speed: 발화 속도 (한국어 토크쇼 현실 반영)
      type: object or enum
      recommended values:
        very_slow: 180~200 자/분
        slow:     200~230 자/분
        normal:   240~270 자/분    # 대부분의 한국 팟캐스트·라디오 자연 속도 (추천 기본값)
        fast:     280~320 자/분
        very_fast: 330+ 자/분
      default: normal (약 255 자/분 기준)
      note: 직접 숫자 입력 가능 (예: 260)
    - paragraph_length: 한 턴(한 사람이 말하는 문단) 목표 글자 수
      default: 280자             # 대화체 자연 호흡 (약 60~70초)
      ranges by style:
        정보전달형: 220~280자
        대화체/수다형: 280~400자
        인터뷰형: 250~350자
  outputs:
    - script: 완성된 대본 (시간대 + 발화자 + 음악/광고 지시 포함)
      format example:
        [00:00 ~ 00:30] 오프닝 음악 + 소개 (30초 FIX)
        MC: 여러분 안녕하세요~ 오늘의 [프로그램명]입니다!
        
        [00:30 ~ 03:00] 주제 소개 + 손님 소개 + 세부 예고 (토크)
        MC: 오늘은 [주제]에 대해 이야기해보려고 해요...
        게스트: 네, 반갑습니다!
        
        [03:00 ~ 03:30] 스팟광고 1 (30초 FIX)
        
        [03:30 ~ 10:00] 소주제 1 토크 (문단 여러 개)
        ...
        
        [마지막] 소주제 4 토크 + 크로징 멘트 + 스팟광고 4 (30초 FIX)
        MC: 오늘도 즐거운 시간이었어요...
        
        [마지막 ~ 57:00] 엔딩 음악 4 (페이드아웃으로 전체 시간 맞춤)
    - char_count: 실제 대본 총 글자 수 (공백 제외 순수 문자 카운트)
    - paragraph_count: 실제 토크 문단(턴) 수
    - time_allocation: 세션별 시간 배분표 (표/리스트 형태)
      example (총 57:00 기준 가정):
        총 목표: 57:00 (3420초)
        - 00:00 ~ 00:30   오프닝 음악 + 소개 (30초 FIX)
        - 00:30 ~ 03:00   주제·손님 소개 (150초, 2~3문단)
        - 03:00 ~ 03:30   스팟광고 1 (30초 FIX)
        - 03:30 ~ 10:00   소주제 1 토크 (390초, 5~6문단)
        - 10:00 ~ 13:30   음악 1 (210초)
        - 13:30 ~ 20:00   소주제 2 토크 (390초)
        - 20:00 ~ 23:30   음악 2 (210초) + 스팟광고 2 (30초)
        - 23:30 ~ 30:00   소주제 3 토크 (390초)
        - 30:00 ~ 33:30   음악 3 (210초) + 스팟광고 3 (30초)
        - 33:30 ~ 55:30   소주제 4 토크 + 크로징 멘트 + 스팟광고 4 (1320초, 다수 문단)
        - 55:30 ~ 57:00   엔딩 음악 4 (90초, 페이드아웃)
        실제 총합: 57:00 (오차 0초 목표)
    - adjustment_log: 글자/시간 부족 시 추가 작업 상세 로그
      example:
        초기 생성: 8,200자 (필요 약 11,000자 대비 -2,800자 부족)
        → 소주제 4 구간에 대화 턴 4개 추가 (+1,400자)
        → 크로징 멘트 확장 + 감사 인사 추가 (+1,450자)
        최종: 11,050자 (오차 +50자, 목표 시간 ±1% 이내)
    - google_docs_export: Google Docs 저장 가이드 및 링크 준비 (Antigravity 연계)
      description: |
        Antigravity IDE에서 다음 프롬프트를 사용해 자동 생성:
        "아래 대본 텍스트를 새 Google Docs 문서로 만들어줘.
        제목: '[프로그램명] - [대주제] 57분 에피소드 원고'
        내용: [script 전체 붙여넣기]
        공유: 나에게만 또는 팀 공유"
        → 에이전트가 Google Drive에 Docs 생성 후 링크 반환
      fallback: 수동으로 script 복사 → docs.google.com에서 새 문서 붙여넣기
      note: Antigravity의 브라우저 자동화 또는 MCP Toolbox 활용 시 90% 이상 자동화 가능
